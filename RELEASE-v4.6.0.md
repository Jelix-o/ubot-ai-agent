# UBot v4.6.0 Release Notes

发布日期：2026-06-06

## 概览

`v4.6.0` 聚焦管理端的可观测性、长任务治理、排期可解释性和画像公开链接治理。版本延续现有 API / 数据结构兼容策略，在不改变业务配置含义的前提下，把耗时操作和模型状态变成可追踪、可回溯的后台能力。

## 主要更新

- 新增任务中心：记录记忆去重、画像生成、模型检测、候选记忆批量审核等耗时管理动作，支持按关键词、类型、状态和群范围筛选。
- 长任务写入任务记录：去重应用、画像生成/刷新、模型连接检测、候选批量审核会生成执行状态、进度、耗时和结果摘要。
- 新增模型健康历史：系统状态页展示最近模型检测来源、结果、延迟和检测时间；总览/健康页检测会写入历史。
- 画像公开链接治理：画像记录新增公开状态、撤销/恢复、过期时间和访问次数；公开页会拦截已撤销、过期或禁用链接，并记录访问次数。
- 群配置新增未来 7 天执行预览：服务端统一计算日报、节日倒计时和群定时任务的未来执行安排，便于核对日期规则和时间范围。
- 新增操作审计页：复用操作日志过滤能力，在管理端按群、动作、关键词和数量查看操作者、目标和详情。
- 任务中心和操作审计新增内联详情面板：可查看完整任务结果、执行时间线、操作动作代码、目标和完整详情，减少表格截断带来的排查成本。
- 管理端截图巡检覆盖 `/tasks`、`/tasks` 详情、`/audit` 与 `/audit` 详情，并更新静态测试锁定 V4.6.0 版本标识、任务中心、操作审计和排期预览入口。
