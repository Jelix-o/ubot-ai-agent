# UBot v4.4.0 Release Notes

发布日期：2026-06-06

## 概览

`v4.4.0` 聚焦管理端可用性、智能工作日排程、模型配置治理、画像公开链接、权限秘钥和系统状态巡检。版本保持现有 API 与数据结构兼容，旧配置里的 `workday` / `holiday` 自动获得智能工作日语义。

## 主要更新

- 日期规则升级为智能工作日：跳过中国大陆法定节假日，包含官方调休上班日，并同步用于日报、节日倒计时和定时提醒。
- 管理端全页面优化：总览搜索改为 Ctrl+K 风格，群下拉只展示可用群，成员卡片改为双列，筛选、详情栏、表格操作列和移动端布局统一收紧。
- 群定时任务改为执行开始时间、执行结束时间和执行间隔，日期规则布局改为上下排列。
- 画像总结支持公开链接展示与复制，群内命令生成和定时任务生成的画像记录都进入后台列表。
- Skills 管理增加恢复备份和复制 Skill，编辑面板随页面滚动，列表操作左对齐。
- 健康状态升级为系统状态，新增模型检测和服务器状态；模型异常按缓存展示，支持按分类检测。
- 系统管理增加模型分类 Tab、单分类单模型选择、模型连接检测、超级管理员秘钥和管理员秘钥配置。
