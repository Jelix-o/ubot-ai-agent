# UBot V1.0.0 系统指令清单

这份文档汇总 V1.0.0 当前支持的群聊指令。后台“指令管理”可以配置内置指令的主命令、别名和启停状态，但权限等级保持内置不可变。

## 权限

- 普通成员：任何群成员。
- 群管理员：当前群 `switcherUserIds`。
- 超级管理员：`config/groups.json` 顶层 `superAdminUserIds`。

群管理员可使用当前群管理命令；超级管理员拥有所有群权限，并可增删群管理员。

## 帮助与状态

### `#功能` / `#帮助` / `#命令`

- 权限：普通成员
- 作用：查看系统功能总览。

### `#功能 <分类>` / `#帮助 <分类>` / `#命令 <分类>`

- 权限：普通成员
- 分类：`对话`、`语音`、`技能`、`实时对话`、`定时任务`、`日报`、`节假日`、`管理员`、`权限`

### `#状态`

- 权限：群管理员 / 超级管理员
- 作用：查看当前群说话状态、当前 skill、实时对话、语音回复、定时任务、日报、节假日、黑名单和管理员数量。
- 说明：默认语音回复开启时，本命令仍返回文字。

### `#健康检查` / `#健康`

- 权限：群管理员 / 超级管理员
- 作用：查看 NapCat 连接、当前技能、允许技能、定时任务、日报、节假日和管理员配置。

### `#服务器`

- 权限：群管理员 / 超级管理员
- 作用：查看主机、Node、PID、运行时长、CPU、内存、工作目录和 NapCat 状态。

### `#操作日志`

- 权限：群管理员 / 超级管理员
- 作用：查看当前群最近 10 条管理员操作。

### `#告警 状态` / `#告警 开启` / `#告警 关闭`

- 权限：群管理员 / 超级管理员
- 作用：查看或切换当前群自动运维告警。

## 对话

### `@机器人 <内容>`

- 权限：普通成员
- 作用：使用当前 skill 生成文字或默认语音回复。
- 上下文：按“当前群 + 当前 QQ”隔离。

### `#对话 清空`

- 权限：普通成员
- 作用：清空自己在当前群的个人上下文。

### `#对话 清空 <QQ号>` / `#对话 清空 @QQ`

- 权限：群管理员 / 超级管理员
- 作用：清空指定成员在当前群的个人上下文。

### `#对话 清空 全部` / `#clear`

- 权限：群管理员 / 超级管理员
- 作用：清空当前群全部成员上下文。

## 语音

### `#语音 <内容>`

- 权限：普通成员
- 作用：先生成 AI 回复，再把回复合成为语音条发送。
- 说明：群配置 `voiceReplyEnabled=false` 时静默拦截语音命令。

### `@机器人 语音说 <内容>`

- 权限：普通成员
- 作用：等价于 `#语音 <内容>`。

### `#语音回复 状态`

- 权限：群管理员 / 超级管理员
- 作用：查看当前群语音功能和默认语音回复状态。

### `#语音回复 开启`

- 权限：群管理员 / 超级管理员
- 作用：开启默认语音回复。普通 AI 回复优先发送语音条，系统指令仍发送文字。

### `#语音回复 关闭`

- 权限：群管理员 / 超级管理员
- 作用：关闭默认语音回复，普通 AI 回复恢复文字。

### `#唱歌 <内容>`

- 权限：普通成员
- 作用：先生成 AI 回复，再用 singing 模式发送语音。
- 说明：只支持 `mimo-v2.5-tts`。如果当前 TTS 模型不支持唱歌，会提示切换模型并回退发送文字。

## Skill 和模型

### `#技能 列表`

- 权限：普通成员
- 作用：查看当前群允许使用的 skill 列表。

### `#技能 切换 <skillId>`

- 权限：群管理员 / 超级管理员
- 作用：切换当前群 skill，并清空该群上下文。

### `#模型状态`

- 权限：普通成员
- 作用：查看当前群聊天回复模型。

### `#模型切换 <modelId|gpt|mimo>`

- 权限：群管理员 / 超级管理员
- 作用：切换当前群聊天回复模型。
- 说明：系统设置里启用的 reply 模型会进入可切换列表；`gpt` 和 `mimo` 为兼容别名。

## 实时对话

### `#实时对话 列表`

- 权限：群管理员 / 超级管理员
- 作用：查看实时对话用户列表和倒计时。

### `#实时对话 添加 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：把某个成员加入实时对话跟踪。

### `#实时对话 移除 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：移除实时对话跟踪成员。

### `#实时对话 间隔 <分钟>`

- 权限：群管理员 / 超级管理员
- 作用：设置机器人最后一次发言后需要安静多久才允许自动接话。

## 记忆和画像

### `#记忆 状态`

- 权限：群管理员 / 超级管理员
- 作用：查看当前群长期记忆启用数量、总数量和待审核候选数量。

### `#知识库 状态`

- 权限：群管理员 / 超级管理员
- 作用：查看当前群 FAQ 启用数量、总数量和关键词检索状态。

### `#昨日画像 <备注/QQ号>`

- 权限：普通成员
- 作用：查看指定成员昨日新增画像总结；省略参数默认查询自己。
- 说明：支持 QQ、系统备注、别名、群名片或昵称查询。多人模糊命中时会提示改用 QQ。

### `#群聊画像 <备注/QQ号>`

- 权限：普通成员
- 作用：汇总指定成员的整体长期群聊画像；省略参数默认查询自己。

## 定时任务

### `@机器人 设置定时任务一个小时提醒群友喝水`

- 权限：普通成员
- 作用：自然语言创建循环提醒任务。

### `#定时任务 列表`

- 权限：普通成员
- 作用：查看当前群定时任务。

### `#定时任务 添加 <规则>`

- 权限：普通成员
- 示例：`#定时任务 添加 每30分钟提醒群友站起来活动`

### `#定时任务 删除 <任务ID>`

- 权限：普通成员
- 作用：删除当前群指定任务。

### `#定时任务 状态`

- 权限：普通成员
- 作用：查看当前群定时任务总开关。

### `#定时任务 开启` / `#定时任务 关闭`

- 权限：群管理员 / 超级管理员
- 作用：恢复或暂停当前群全部定时任务触发，不删除任务。

## 群聊日报

### `#日报` / `#日报 状态`

- 权限：普通成员
- 作用：查看日报是否开启和发送时间。

### `#日报 发送`

- 权限：群管理员 / 超级管理员
- 作用：立即手动发送一次日报。

### `#日报 开启` / `#日报 关闭`

- 权限：群管理员 / 超级管理员
- 作用：开启或关闭群聊日报。

### `#日报 时间 <HH:mm>`

- 权限：群管理员 / 超级管理员
- 作用：设置日报自动发送时间。

## 节假日倒计时

### `#节假日` / `#节假日 查看`

- 权限：普通成员
- 作用：查看最近已公布的中国节假日倒计时。

### `#节假日 状态`

- 权限：普通成员
- 作用：查看节假日倒计时开关、时间和当前倒计时。

### `#节假日 发送`

- 权限：群管理员 / 超级管理员
- 作用：立即手动发送一次节假日倒计时。

### `#节假日 开启` / `#节假日 关闭`

- 权限：群管理员 / 超级管理员
- 作用：开启或关闭自动节假日倒计时。

### `#节假日 时间 <HH:mm>`

- 权限：群管理员 / 超级管理员
- 作用：设置自动发送时间。

## 管理员、闭嘴和黑名单

### `#管理员` / `#管理员 列表`

- 权限：群管理员 / 超级管理员
- 作用：查看本群管理员和超级管理员列表。

### `#管理员 添加 <QQ号>`

- 权限：超级管理员
- 作用：添加当前群管理员。

### `#管理员 移除 <QQ号>`

- 权限：超级管理员
- 作用：移除当前群管理员。

### `#闭嘴`

- 权限：群管理员 / 超级管理员
- 作用：当前群停止普通对话、语音、实时对话、复读和关键词主动触发。
- 保留：总结、日报、节假日、定时任务提醒和管理命令。

### `#说话`

- 权限：群管理员 / 超级管理员
- 作用：恢复当前群普通回复能力。

### `#拉黑 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：拉黑当前群指定成员。被拉黑成员发言仍进入日报和聊天总结，但机器人不回复。

### `#拉黑 解除 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：解除当前群黑名单。

## 行为规则

- 普通对话只在 `@机器人`、关键词触发或实时对话条件满足时发生。
- 系统指令不会受默认语音回复影响，始终文字返回。
- 普通 `#语音` 成功时只发送语音，不额外补文字。
- `#唱歌` 使用 `(唱歌)` 标签，只允许 `mimo-v2.5-tts`。
- AI 正文里的第三方 `@QQ`、`@名字` 或 CQ at 会被降级成普通文本，避免误触发第三方提醒。
- 同一个群最多并发处理 10 条消息，超过后排队处理。
